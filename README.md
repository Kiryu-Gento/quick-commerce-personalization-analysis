## 분석 목표

고객 소비 규모보다 **구매 리듬과 이용 패턴**이 구독 전환에 영향을 미친다는 가설을 검증하고,
휴면 고객의 단계적 복귀 전략까지 도출하는 것이 목표.

1. **구독 전환 요인 분석** — 구독 vs 비구독 소비 패턴 비교, t-test 유의성 검증
2. **시기별 구매 트렌드** — 월별·계절별 인기 상품 도출, FP-Growth 연관 분석
3. **휴면 고객 분석** — RFM 기반 휴면 정의, 위험군 세분화, 리텐션 전략 수립

## 분석 흐름 요약

### 1. 구독 전환 요인 분석
- 구독자 vs 비구독자 소비 지표 비교 (총구매금액, 구매횟수, 평균구매금액 등)
- t-test로 통계적 유의성 검증
- **결과**: 평균 소비금액은 통계적으로 유의(p=0.0066)하나 실용적 차이는 작음
  → 핵심 변수는 **소비 규모가 아닌 월평균 구매횟수 패턴**으로 판단

### 2. 시기별 구매 트렌드
- 월별·계절별 상품 대분류 주문 빈도 지수 계산
- FP-Growth 기반 상품 연관 분석 (지지도, 신뢰도, 향상도)
- **결과**: 채소류·두부·유정란 간 높은 연관성 → 묶음 상품 및 추천 전략 도출

### 3. 휴면 고객 분석
- RFM 기반 휴면 정의 (60일 이상 미구매)
- 구매 빈도 기준 위험군 분할 (High / Mid / Low)
- 연령 구간별 휴면율 분석
- **결과**: 전체 고객 중 휴면 비율 37.7%, 그중 비구독자 83.55% 식별

## 핵심 인사이트

| 분석 항목 | 핵심 발견 | 개선안 |
|---|---|---|
| 구독 전환 요인 | 소비 규모 차이는 작고 **이용 패턴**이 전환 결정 | 구매 리듬 형성 고객 대상 구독 제안 노출 |
| 적립금/포인트 | 구독 전환에 통계적으로 **비유의** (p > 0.05) | 금전 혜택 대신 **이용 편의성** 중심 구독 설계 |
| 월별 인기상품 | 계절·월별 구매 패턴 존재 | 월별 인기 상품관 운영, 리마인드 푸시 |
| 연관 분석 | 채소·두부·유정란 간 높은 연관성 | 장바구니 바텀시트 도입, 묶음 상품 개발 |
| 휴면 비율 | 전체 고객의 약 37.7% 휴면 상태 | 30일/45일/60일 단계적 개입 |
| 휴면 고객 특성 | 비구독자 비율 높음, 20·30대 휴면율 상대적 높음 | High/Mid 위험군 우선 개입으로 ROI 극대화 |

## 제안 전략

**구독자 전환 전략**
1. 월별 인기 상품관 운영 — 시기별 인기 상품 메인 배너 배치
2. 리마인드 푸시 알림 — 시기별 구매 이력 고객 대상 발송
3. 장바구니 바텀시트 도입 — 연관분석 기반 함께 구매할 상품 제안
4. 묶음 상품 개발 (Bundling Marketing) — 두부+콩나물+유정란 패키지 등

**휴면 고객 집중 전략**
- 구매 공백 기반 단계적 개입: 30일 리마인드 → 45일 혜택 제안 → 60일 복귀 캠페인
- **High 위험군**: 1만원 쿠폰 + 무료배송 (푸시+카톡, 오후 8시)
- **Mid 위험군**: 5천원 쿠폰 (푸시, 오후 8시)
- **Low 위험군**: 콘텐츠 중심 관계 유지 (월 1회)
